googleClient

CLI tool for Google Workspace domain administrators to safely audit user Google Drives using Domain-Wide Delegation. Read-only by design — no content modifications, no access-time updates — built for migration planning and file governance at scale.

dmarcParser

Interactive CLI and REPL for ingesting and analyzing DMARC aggregate (RUA) reports. Instantly reveals which IP addresses and senders pass or fail SPF/DKIM/DMARC, surfaces impersonation attempts, and diagnoses email deliverability issues through a SQLite-backed per-client database.

Python CLI / REPL SQLite Email Security DMARC

mochi-gallery

AI art station that transforms Mochimo Blockchain blocks into high-fidelity poster art. Extracts each block's unique "Haiku" value, uses Google Gemini to interpret artistic mood, and deploys Google Imagen for illustration rendering — complete with a local web gallery interface.
